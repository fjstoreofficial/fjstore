# FJ Store - Toko Online

Selamat datang di **FJ Store**, toko online sederhana untuk produk fashion.  
Website ini menggunakan HTML dan CSS untuk menampilkan produk, jam kerja, serta tombol WhatsApp untuk pemesanan.

---

## Struktur Repository



---

## Fitur Website

1. **Halaman Toko**  
   - Menampilkan produk dengan **deskripsi di bawah gambar**.
   - Kotak produk compact, maksimal **2 produk per baris** di HP maupun laptop.
   - Background halaman terlihat jelas.
   
2. **Jam Kerja**  
   - Ditampilkan di kotak kecil: `09:00 - 21:00`.

3. **Tombol WhatsApp**  
   - Mengambang di kanan bawah layar.
   - Template pesan otomatis:
     ```
     HALO FJ STORE

     Nama:
     Alamat:
     Ukuran:
     Pembayaran melalui:
     ```

---

## Cara Menggunakan

1. Buka `index.html` untuk halaman utama.  
2. Klik menu atau link ke `toko.html` untuk melihat produk.  
3. Klik tombol **Pesan via WA** untuk langsung chat ke admin dengan template pesan.  

---

## Catatan

- Pastikan **gambar produk** dan **background** berada di folder yang sama dengan HTML.  
- Gambar produk bisa menggunakan format **PNG** atau **WebP**, tidak harus JPG.  
- Deskripsi produk diletakkan **di bawah gambar**, agar layout rapi di semua perangkat.
